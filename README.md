# OpenGradient Quiz Game 🧠

A Kahoot/Quizizz-style interactive quiz game about the **OpenGradient AI project** — test your knowledge of decentralized AI inference, on-chain ML models, and the OpenGradient ecosystem!

## Features

- **15 Questions** covering the OpenGradient platform, its technology, tokenomics, and vision
- **Countdown Timer** — answer quickly to maximize your score
- **Scoring System** — points are awarded based on correctness and speed
- **Confetti Effects** — celebrate correct answers and high scores with visual flair
- Clean, responsive UI inspired by popular quiz platforms

## How to Play

1. Open `index.html` in any modern web browser
2. Click **Start Quiz** to begin
3. Select your answer before the timer runs out
4. Try to get a perfect score!

## About OpenGradient

[OpenGradient](https://opengradient.ai) is a decentralized AI inference network that enables trustless, on-chain execution of machine learning models. It bridges AI and blockchain, allowing developers to deploy and run ML models in a verifiable, permissionless way.

## Tech Stack

- Pure HTML, CSS, and JavaScript (no dependencies)
- Single-file application — just open and play!
